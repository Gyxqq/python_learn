## while 循环
```python
while 某个条件 :
    代码块
    代码块
循环外的代码
```
- python的循环，判断等结构是由代码缩进实现的，如:
```python   
g=0
c=0
while g<20 :
        g=g+1
        c+=g  ##这里退出循环
print(g,c)
```
while 后面跟的条件在满足时才会进入循环

## for 循环
```python
for XXX in XXX :
    XXX
    XXX
循环外的代码
```
- for循环使用迭代对象，如字符串或者列表
```python
str1='abcde'
for abc in str1:
    print(abc)
```
执行效果如下
```shell
a
b
c
d
e
```
或者
```python
for c in [1,4,5,6,7]:
    print(c)
```
效果
```shell
1
4
5
6
7
```
或者能利用range函数生成的列表
```python
for a in range(1,7):
    print(a)
```
## **break语句**
在循环中加入break会提前结束循环
## ep:
```python
a=0
while a <12 :
    if a==4:
        break
    print(a)
    a=a+1
```
执行效果
```shell
0
1
2
3
```
- 在for 循环里面也是一样的
## **continue语句**   
continue语句可跳过当次循环中剩下的 语句，但是不会跳出循环  
```python
for a in range(1,5):
    if a==3 :
        continue
    print(a)
    a=a+1
```
效果如下
```shell
1
2
4
```
